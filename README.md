# 🚀 Overview

🎥 **Project Demonstration**

[▶ Watch the Nirvana Smart-Chain Demo](https://drive.google.com/file/d/15_sGRl-Bk8546jOyXU9_MJ-yy4t-Uqm4/view?usp=sharing)

**Nirvana Smart-Chain** is a blockchain-powered pharmaceutical supply chain platform designed to eliminate **drug counterfeiting**, increase **transparency**, and ensure **end-to-end traceability** of medicines.

By integrating **blockchain technology, smart contracts, and real-time verification**, the platform enables manufacturers, distributors, pharmacies, and regulators to verify drug authenticity at every stage of the supply chain.

The system ensures that every transaction in the drug lifecycle is **tamper-proof, time-stamped, and publicly verifiable**, creating a trustworthy ecosystem for pharmaceutical distribution.

---

# 🎯 Problem Statement

Counterfeit medicines are a **major global healthcare crisis**.

They:

* Endanger patient lives
* Cause massive financial losses to pharmaceutical companies
* Undermine trust in healthcare systems

Traditional pharmaceutical supply chains suffer from:

* Lack of transparency
* Weak traceability
* Tamper-prone records
* No real-time verification mechanisms

These limitations allow counterfeit drugs to enter the market.

---

# 💡 Our Solution

**Nirvana Smart-Chain** introduces a **decentralized pharmaceutical supply chain verification system** powered by blockchain.

Key capabilities include:

🔐 **Immutable Blockchain Records**
Every drug batch and transaction is permanently stored on the blockchain.

🧾 **Smart Contract Drug Registration**
Manufacturers register drug batches using blockchain smart contracts.

🔍 **Real-Time Drug Traceability**
Every stakeholder can track drug movement from origin to destination.

🌐 **Multi-Stakeholder Access**
Different roles interact with the system:

* Manufacturer
* Wholesaler / Distributor
* Retailer / Pharmacy
* Consumer
* Regulator

🪙 **Tokenized Ecosystem**
ERC-20 tokens enable transactions, incentives, and ecosystem interactions.

# 🛠️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Sourjya-Saha/Nirvana-Smart-Chain-NSC.git
cd Nirvana-Smart-Chain-NSC
```

---

# 2️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:3000
```

---

# 3️⃣ Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

Backend runs on:

```
http://localhost:5000
```

---

# 4️⃣ Smart Contract Deployment

Compile contracts:

```bash
cd frontend
npx hardhat compile
```

Deploy to Polygon Amoy:

```bash
npx hardhat deploy --network amoy
```

Make sure you have **Polygon Amoy configured in MetaMask**.

---

# 🔐 Environment Variables

Create a `.env` file:

```env
DATABASE_URL=your_mysql_url
PRIVATE_KEY=your_wallet_private_key
RPC_URL=polygon_amoy_rpc_url
```
